Place your branding files in this folder with these exact names:

1) anomalie-logo-full.png
   - Used in hero section
   - Recommended: PNG, transparent background
   - Recommended width: 1400px to 2200px (landscape)

2) anomalie-logo-icon.png
   - Used in header + footer
   - Recommended: PNG, square
   - Recommended size: 512x512

3) anomalie-favicon-32.png
   - Browser tab favicon
   - Required size: 32x32 PNG

4) anomalie-favicon-192.png
   - High-resolution favicon/app icon
   - Required size: 192x192 PNG

5) anomalie-apple-icon.png
   - iOS home-screen icon
   - Recommended size: 180x180 PNG

After adding/replacing files, restart dev server if needed.
